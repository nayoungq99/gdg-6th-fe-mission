HTML 기본 문법









- JavaScript 기본 문법
:블록 유효범위를 생성하지 않는다.

- 데이터 타입과 변수
:원시 타입(변경 불가능한 값, 값에 의한 전달) -> 문자열
 - undefined -> 선언 되었지만 값 할당하지 않은 변수 접근, 존재하지 않는 객체에 접근할 경우 반환되는 값. (자바스크립트 엔진에 의해 초기화)
 - null 타입 확인할 때 일치연산자 사용.
 - 동적 타입 언어라 자동으로 타입이 결정됨. 같은 변수에 여러 타입의 값을 할당할 수 있고 이를 동적 타이핑이라고 부름.
 - 모든 선언문 호이스팅 됨. (모든 선언문이 해당 scope 선두로 옮겨진 것처럼 동작) 
   선언 단계, 초기화 단계, 할당 단계, 함수 레벨 스코프(함수 내에서만 유효)

- 표현식과 연산자
: 쉼표 연산자(왼쪽 피연산자부터 차례로 평가, 마지막 피연산자 반환)
  할당 연산자(좌항에 있는 변수에 할당. 부수 효과가 있다) -> =, +=, -=, *=, /=, %=

- 제어문
: 블록문을 하나의 단위로 취급함. 세미콜론 붙이지 않는다.
  continue 문은 반복문의 코드 블록 실행을 현 지점에서 중단. if문 내에서 실행해야 할 코드가 길다면 continue문을 사용하는 것이 가독성 면에서 좋음.
