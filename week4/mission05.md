# 팀원 피드백
## 1. 윤소연 님 피드백
- **피드백**: 단순히 이론만 나열한 것이 아니라 데이터를 어디서 요청하고 어디서 저장할지, 조건부 렌더링을 통한 관리자 페이지 구현 등을 기록한 점이 매우 인상적이에요... 특히 3주차 미션에서 API 함수를 모듈화하여 itemApi.js로 분리하고 이를 컴포넌트에서 호출하는 흐름을 깨닫는 과정이 더 대단해보엿어요...
- **느낀 점**: React 컴포넌트의 생애 주기, useEffect를 이용한 데이터 Fetching, 모듈화를 통한 API 호출 관리 등 실무적인 코드 예시를 적절히 포함된다면 글의 신뢰성이 높아보인다는 점을 깨달았다.

## 2. 조현 님 피드백
- **피드백**: 단순히 문법을 배운 것이 아니라 '어디서 데이터를 요청하고, 상태를 어디서 관리할지'에 대해 끊임없이 고민하신 점과 매 미션을 하면서 error를 마주했던 경험을 기록하신 점이 인상 깊습니다..!
- **느낀 점**: 1주차의 컴포넌트 구조 이해부터 2주차의 React Router와 상태 관리, 그리고 3주차의 API 통신과 비동기 처리까지, 프론트엔드 개발의 핵심 과정이 잘 정리되어있어 읽기 좋았다.

## 3. 박준서 님 피드백
- **피드백**: 프로젝트의 흐름을 본질적으로 이해하기 위해 관련 도서를 직접 학습하시는 모습에서 개발자로서의 진정성을 느꼈습니다... 파편화된 정보들을 논리적으로 재구성하여 핵심 위주로 깔끔하게 정리해주신 덕분에, 독자 입장에서 훨씬 효율적으로 정보를 습득할 수 있었어요!
- **느낀 점**: "도서 구매를 마다하지 않는 주도적인 학습 방식과 코드의 근본적인 이해를 지향하는 점이 멋있어 보였다. AI의 조력에 의존하기보다 본인만의 로직으로 코드를 완성해 나가는 '손코딩'의 가치를 추구하는 모습을 닮고 싶다!

# 배워야할 점!
AI 도구의 효율성에 의존해 '딸깍'하며 결과물을 만들어내는 방식에서 벗어나 내 힘으로 직접 로직을 구현하는 '손코딩'의 가치를 추구하는 모습을 닮고 싶다. 또 공식 문서를 기반으로 학습한 개념을 실제 프로젝트에 적용하고 그 과정에서 마주한 에러를 회피하지 않고 끝까지 원인을 찾아내어 기록하는 끈기를 닮아 성장하는 데 기회로 삼을 것이다.
뿐만 아니라 방대한 개발 지식들을 논리적인 구조에 따라 일목요연하게 정리하는 능력이 뛰어난 팀원도 있는 것 같아... 블로그에 작성하는 힘도 더 발전시키고 싶다...
